# AI-Powered Portfolio Optimization System

An end-to-end data science project that predicts stock returns using machine learning and optimizes portfolio allocation using Modern Portfolio Theory. The system is deployed as an interactive web application using Streamlit.

---

# Tech Stack

- **Python**
- **pandas** – Data manipulation  
- **numpy** – Numerical computations  
- **scikit-learn** – Machine learning model  
- **SciPy** – Optimization  
- **Streamlit** – Web application  
- **yfinance** – Data source  


---

# Features

- Fetches historical stock data from Yahoo Finance  
- Performs feature engineering (returns, volatility, momentum, moving averages)  
- Trains a Machine Learning model (Random Forest) to predict future returns  
- Optimizes portfolio allocation using Sharpe Ratio maximization  
- Interactive dashboard built with Streamlit  
- Displays optimal asset allocation and investment breakdown
